Python 2.7 will reach end of life in...

What's all this, then?

Python 2.7, and by extension the entire Python 2 family, will not be maintained past 2020. Since no official date has been given, this clock counts down until April 12th, 2020, which will be roughly the time of the 2020 PyCon. No one has said that PyCon 2020 will be the official death-date, but it seems like a good idea. Maybe we'll have a wake, and a party. In any case, the time to update to Python 3 is now, and there are lots of resources to help.

Python 2, thank you for your years of faithful service.

Python 3, your time is now.
